Output: A running server that accepts TCP connections, registers accounts, authenticates logins, and handles clean logout.

2. Create `server/src/auth/hash.rs` with two public functions:

- `pub fn hash_password(password: &str) -> anyhow::Result&lt;String&gt;` -- Uses `Argon2::default()` (Argon2id v19, m=19456, t=2, p=1). Generates salt with `SaltString::generate(&amp;mut OsRng)`. Returns the PHC string format (`$argon2id$v=19$...`).

- `pub fn verify_password(password: &str, phc_hash: &str) -> anyhow::Result&lt;bool&gt;` -- Parses PHC hash string, verifies password bytes against it. Returns `Ok(true)` if match, `Ok(false)` if wrong password.

Both functions are synchronous (intended to be called via `tokio::task::spawn_blocking`). Import from `argon2::{Argon2, PasswordHash, PasswordHasher, PasswordVerifier}` and `argon2::password_hash::{rand_core::OsRng, SaltString}`.

3. Create `server/src/auth/session.rs` with these functions using raw sqlx queries (NOT query! macros -- to avoid compile-time DB requirement in this phase):

- `pub async fn create_session(pool: &SqlitePool, account_id: &str, ttl_secs: i64) -> anyhow::Result&lt;String&gt;` -- Generates UUID v4 token, inserts into `sessions` table with `created_at = unixepoch()` and `expires_at = unixepoch() + ttl_secs`. Returns the token string.

- `pub async fn validate_session(pool: &SqlitePool, token: &str) -> anyhow::Result&lt;Option&lt;String&gt;&gt;` -- Queries sessions table for token where `expires_at > unixepoch()`. Returns `Some(account_id)` if valid, `None` if expired or not found.

- `pub async fn delete_session(pool: &SqlitePool, token: &str) -> anyhow::Result&lt;()&gt;` -- Deletes the session row. Used for logout (AUTH-08).

- `pub async fn register_account(pool: &SqlitePool, username: &str, password_hash: &str) -> anyhow::Result&lt;String&gt;` -- Generates UUID v4 account id, inserts into `accounts` table. Returns account id. Must handle UNIQUE constraint violation on username and return a descriptive error.

- `pub async fn lookup_account(pool: &SqlitePool, username: &str) -> anyhow::Result&lt;Option&lt;(String, String)&gt;&gt;` -- Returns `Some((account_id, password_hash))` if found, `None` if not.

Use `sqlx::query()` with `.bind()` (runtime query, not compile-time macro) for all queries. Use `uuid::Uuid::new_v4().to_string()` for IDs. Import `sqlx::SqlitePool`.

2. Create `server/src/net/listener.rs` with the TCP accept loop:

- `pub async fn run_listener(addr: &str, state: AppState) -> anyhow::Result&lt;()&gt;` -- Binds `TcpListener` to addr, logs "listening on {addr}". In a loop: accepts connections, logs peer_addr, clones state, calls `tokio::spawn(handle_connection(socket, peer_addr, state))`. Each spawn is independent -- satisfies NETW-01. On accept error, log warning and continue (do not crash the server).

- `async fn handle_connection(socket: TcpStream, peer_addr: SocketAddr, state: AppState) -> anyhow::Result&lt;()&gt;` -- Splits socket into `OwnedReadHalf` and `OwnedWriteHalf` using `socket.into_split()`. Creates a `ConnectionActor` and calls its `run()` method. On completion, logs "connection closed" with peer_addr.

Define `AppState` as:
```rust

use std::sync::Arc;

#[derive(Clone)]
pub struct AppState {
    pub db: SqlitePool,
    pub session_ttl_secs: i64,
}
```

Put `AppState` in `server/src/net/listener.rs` or a shared location importable by both listener and actor.

3. Create `server/src/session/mod.rs`:

4. Create `server/src/session/actor.rs` with `ConnectionActor`:

```rust
pub struct ConnectionActor {
    reader: OwnedReadHalf,
    writer: OwnedWriteHalf,
    state: AppState,
    session_token: Option<String>,  // None until authenticated
    account_id: Option<String>,     // None until authenticated
}
```

Implement:
- `pub fn new(reader, writer, state) -> Self`
- `pub async fn run(&mut self) -> anyhow::Result&lt;()&gt;` -- Main loop: reads 4-byte LE length prefix from reader, then reads that many payload bytes, decodes as `ClientMsg` using `protocol::codec::decode_message`, dispatches to `handle_message`. On EOF or error, break. Before returning, if authenticated, call `cleanup()`.

- `async fn handle_message(&mut self, msg: ClientMsg) -> anyhow::Result&lt;()&gt;` -- Match on ClientMsg variants:
  - `Register { username, password }`: Call `spawn_blocking` with `hash_password(&password)`. Then call `register_account(pool, &username, &hash)`. On success, send `ServerMsg::RegisterOk { account_id }`. On UNIQUE violation (username taken), send `ServerMsg::Error { code: UsernameTaken, message }`.
  - `Login { username, password }`: Call `lookup_account(pool, &username)`. If None, send InvalidCredentials error. If Some, call `spawn_blocking` with `verify_password(&password, &hash)`. If false, send InvalidCredentials. If true, call `create_session(pool, &account_id, ttl)`. Store session_token and account_id in self. Send `ServerMsg::LoginOk { session_token }`.
  - `Logout`: If authenticated, call `delete_session(pool, &token)`. Clear session_token and account_id. Send `ServerMsg::LogoutOk`.
  - `Ping`: Send `ServerMsg::Pong`.

- `async fn send(&mut self, msg: ServerMsg) -> anyhow::Result&lt;()&gt;` -- Encode with `protocol::codec::encode_message(&msg)`, write all bytes to writer with `writer.write_all(&bytes).await`.

- `async fn cleanup(&mut self)` -- If session_token is Some, delete the session from DB (graceful disconnect = auto-logout per AUTH-08).

- `async fn read_frame(&mut self) -> anyhow::Result&lt;Option&lt;Vec&lt;u8&gt;&gt;&gt;` -- Read exactly 4 bytes for length prefix. If EOF on first byte, return None (clean disconnect). Convert to u32 LE. Validate <= 64KiB. Read exactly that many bytes into a Vec. Return Some(bytes).
