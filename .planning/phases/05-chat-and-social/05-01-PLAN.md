# Plan 05-01: Chat Channels, Player Inspection, and Descriptions

**Phase:** 05-chat-and-social
**Plan:** 1 of 1
**Scope:** SOCL-01 through SOCL-06

## Goal

Players can communicate via local (say, emote, whisper) and global (gossip) channels with IC/OOC distinction. Players can toggle channels, inspect other players, and set visible character descriptions.

## Task 1: Protocol Messages and Chat Commands

### Protocol additions (extend world namespace)

Add to `protocol/src/world.rs` ClientMsg:
- `Say { text }` — local IC chat (room only)
- `Emote { text }` — local IC emote (room only)
- `Whisper { target, text }` — private message to a player in the same room
- `Gossip { text }` — global OOC chat (all online players)
- `ToggleChannel { channel }` — toggle a chat channel on/off
- `LookAt { target }` — inspect another player (SOCL-05)
- `Description { text }` — set visible character description (SOCL-06)

Add to `protocol/src/world.rs` ServerMsg:
- `ChatMessage { channel, sender, text }` — incoming chat message
- `EmoteMessage { sender, text }` — incoming emote
- `WhisperMessage { from, text }` — incoming whisper
- `LookAtResult { name, race, class, level, description, bio, equipped }` — player inspection
- `DescriptionOk`
- `ChannelToggled { channel, enabled }` — confirmation of channel toggle

### Server changes

- New `server/src/social/mod.rs` + `server/src/social/commands.rs`
- Global gossip broadcast: use a server-wide `broadcast::Sender<ChatEvent>` (not per-room)
- Local say/emote: use existing room broadcast channels
- Whisper: find target player in same room, send directly
- Channel toggles: per-character HashMap<String, bool> in memory (persisted to DB)
- Player inspection: query target character from DB, send equipped gear + stats
- Character description: new column `description` on characters table

### Migration

**`server/migrations/008_social.sql`**:
- Add `description` column to characters
- `channel_toggles` table: character_id, channel, enabled

### Integration tests

- `test_say_local_chat` — say is received by same-room player, not by distant player
- `test_gossip_global` — gossip received by all online players
- `test_channel_toggle` — toggle gossip off, don't receive gossip messages
- `test_look_at_player` — inspect shows gear, bio, description
- `test_set_description` — set and verify description via look at

### TUI client update

- Parse say/emote/whisper/gossip commands
- Display chat with IC/OOC markers: `[IC] Grok says: ...` vs `[OOC] Grok gossips: ...`
- Handle ChatMessage, EmoteMessage, WhisperMessage in game log
