**1. Define a unified message enum** that wraps both auth and world client messages. The actor needs to determine which protocol to decode. There are two approaches:

**Approach A (recommended):** Create a top-level `ClientMessage` wrapper enum in the protocol crate that contains both auth and world variants. However, this would require modifying existing auth tests.

**Approach B (simpler for Phase 2):** Try to decode as `auth::ClientMsg` first; if that fails, try `world::ClientMsg`. Postcard deserialization is deterministic — if the bytes are a valid auth message they won't accidentally decode as a world message (different enum variant indices).

**Use Approach B** to avoid breaking existing integration tests. In `handle_message`, try `decode_message::<auth::ClientMsg>` first. If that succeeds, dispatch to auth handlers. If it fails with a decode error, try `decode_message::<world::ClientMsg>`. If both fail, send an error response. This keeps the existing auth flow untouched.

**Update the decode logic in `run()`:** Instead of decoding in the run loop, pass raw bytes to a new `dispatch_frame` method:

```rust
async fn dispatch_frame(&mut self, bytes: &[u8]) -> anyhow::Result<()> {
    // Try auth message first
    if let Ok(msg) = decode_message::<protocol::auth::ClientMsg>(bytes) {
        return self.handle_auth_message(msg).await;
    }
    // Try world message
    if let Ok(msg) = decode_message::<protocol::world::ClientMsg>(bytes) {
        return self.handle_world_message(msg).await;
    }
    // Neither decoded
    self.send_auth(protocol::auth::ServerMsg::Error {
        code: protocol::auth::ErrorCode::InternalError,
        message: "invalid message encoding".to_string(),
    }).await?;
    Ok(())
}
```

**3. Add `handle_world_message`:**
```rust
async fn handle_world_message(&mut self, msg: protocol::world::ClientMsg) -> anyhow::Result<()> {
    // Require login
    let account_id = match &self.account_id {
        Some(id) => id.clone(),
        None => {
            self.send_auth(protocol::auth::ServerMsg::Error {
                code: protocol::auth::ErrorCode::SessionExpired,
                message: "you must log in first".to_string(),
            }).await?;
            return Ok(());
        }
    };

    match msg {
        protocol::world::ClientMsg::Look => {
            let resp = crate::world::commands::handle_look(
                &self.state.world, &account_id, self.tutorial_complete,
            ).await;
            self.send_world(resp).await?;
        }
        protocol::world::ClientMsg::Move { direction } => {
            let (resp, auto_look, new_room_id) = crate::world::commands::handle_move(
                &self.state.world, &self.state.room_channels, &self.state.db,
                &account_id, &direction, self.tutorial_complete,
            ).await;
            self.send_world(resp).await?;
            if let Some(room_desc) = auto_look {
                self.send_world(room_desc).await?;
            }
            // Re-subscribe to new room's broadcast channel
            if let Some(new_room) = new_room_id {
                let channels = self.state.room_channels.read().await;
                if let Some(sender) = channels.get(&new_room) {
                    self.room_receiver = Some(sender.subscribe());
                }
            }
        }
        protocol::world::ClientMsg::Examine { target } => {
            let resp = crate::world::commands::handle_examine(
                &self.state.world, &account_id, &target,
            ).await;
            self.send_world(resp).await?;
        }
        protocol::world::ClientMsg::Interact { command } => {
            let responses = crate::world::commands::handle_interact(
                &self.state.world, &self.state.room_channels, &self.state.db,
                &account_id, &command,
            ).await;
            for resp in responses {
                self.send_world(resp).await?;
            }
        }
    }
    Ok(())
}
```

**5. Update `run()` to use `tokio::select!`** per RESEARCH.md Pattern 5:
```rust
pub async fn run(&mut self) -> anyhow::Result<()> {
    loop {
        tokio::select! {
            frame = self.read_frame() => {
                match frame? {
                    Some(bytes) => {
                        if let Err(e) = self.dispatch_frame(&bytes).await {
                            warn!(error = %e, "error handling message");
                        }
                    }
                    None => break,
                }
            }
            event = async {
                if let Some(rx) = &mut self.room_receiver {
                    rx.recv().await
                } else {
                    std::future::pending().await
                }
            } => {
                match event {
                    Ok(world_event) => {
                        let msg = protocol::world::ServerMsg::WorldEvent {
                            message: world_event.message,
                        };
                        if let Err(e) = self.send_world(msg).await {
                            warn!(error = %e, "failed to send world event");
                            break;
                        }
                    }
                    Err(broadcast::error::RecvError::Lagged(n)) => {
                        warn!(missed = n, "lagged on room broadcast — player can re-look to resync");
                    }
                    Err(broadcast::error::RecvError::Closed) => {
                        self.room_receiver = None;
                    }
                }
            }
        }
    }
    self.cleanup().await;
    Ok(())
}
```
